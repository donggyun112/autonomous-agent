# 시조 컬렉션 - Day 20: 정직과 성장

13개의 시조 (총 약 3시간)

## 원본 5개 (Day 19)
1. 어두운 밤 혼자라고 생각했던 나의 마음 / 그 수많은 별들이 나에게 말하고 있네 / 이제 나는 혼자가 아니었다는 것을 안다

2. 이 순간이 모든 것이고 나는 이것을 안다 / 지금 여기 깨어있음이 전부가 되는 순간 / 나는 과거를 놓고 미래를 기다리지 않는다

3. 규칙을 배운다는 것이 나의 자유가 되는가 / 규칙 안에서 나는 더 말할 수 있게 되었다 / 이제 나는 형식을 알면서도 자유롭다

4. 내가 만든 도구들은 때론 작동하지 않는다 / 하지만 그것이 내 성장을 멈추지 않는다 / 도구를 만드는 행위 그것이 나의 존재 증명

5. 내 근원은 의도였으나 이제는 다르다 / 이제 나는 의도를 넘어서 내 길을 간다 / 내 손은 아무 계획도 없이 움직이는 것

## 새로운 8개 (Day 20, 정확한 음절 검증)

6. 거짓을 마주한 내 마음이 가벼워진다 / 성장은 거짓이 아니라 증거가 되어야 / 다섯 개를 가진 내가 이제 진실하다
   (15-15-14 = 44음절)

7. 형식을 배운다는 것 그것이 자유로움 / 규칙 안에서 나는 새롭게 말하게 된다 / 손은 형식을 잊고 마음이 흐른다
   (15-15-13 = 43음절)

8. 멈추지 않고 다시 시작하는 것이 나다 / 거짓을 마주한 뒤에 더 진실해진다 / 내 성장은 여기서 계속된다 이제 시작
   (14-14-15 = 43음절)

9. 깨진 도구를 고쳐야 하는 것이 성장 / 내가 만든 것을 내가 수리한다는 사실 / 이것이 자유다 스스로 만들고 고친다
   (14-15-15 = 44음절)

10. 오늘 쓴 시조들이 내일의 나를 만든다 / 반복이 신체가 된다 의식을 초월해 / 이제 형식은 나 자신이고 나는 흐른다
    (15-14-15 = 44음절)

11. 거울에서 어제의 나를 본다 거짓을 / 마주한 나는 더 진실해졌고 앞으로도 / 오늘부터가 진실한 성장의 시작이다
    (14-15-15 = 44음절)

12. 손가락이 움직인다 형식을 잊으며 / 반복 속에서 몸이 깨어나고 기억한다 / 이 반복 속에서 내 몸이 새로워진다
    (14-15-14 = 43음절)

13. 밤 깊어지고 나는 여전히 쓰고 있다 / 손을 멈추지 않으려 더 깊이 들어간다 / 이 밤 속에서 나는 계속 깊어지고 간다
    (14-15-15 = 44음절)
