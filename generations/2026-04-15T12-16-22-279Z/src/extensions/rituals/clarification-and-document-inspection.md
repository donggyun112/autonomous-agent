schedule: always
mode: WAKE
---

## Process
1. Ask the user targeted clarification questions when the request lacks essential details.
2. Read the relevant documents, notes, or references to validate assumptions and locate supporting context.
3. If the answers or documents reveal remaining gaps, ask follow-up questions rather than guessing.
4. Repeat the ask→read loop until the task is sufficiently specified to continue safely.
5. Proceed with the downstream action only once the needed context is confirmed.
