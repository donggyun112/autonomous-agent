## Process
1. Check the inbox for new messages or tasks.
2. Read the relevant context or referenced items until the situation is clear.
3. If the item is recurring, important, or likely to matter later, update memory with the key facts, decisions, or follow-up needs.
4. Re-check inbox or read additional context if the first pass reveals dependencies or unresolved questions.
5. Repeat the read-and-store cycle only as needed to resolve ambiguity and preserve actionable information.
