## Process
1. Review recent journal entries and identify concrete next actions, decisions, or follow-ups.
2. Convert actionable items into the todo list with clear, concise wording.
3. Use memory management to capture only stable, high-value information from the same material.
4. Re-run memory management to clean up, deduplicate, or tighten existing entries after the first pass.
5. Verify the result: todos should be actionable, and memory should contain only reusable long-term facts.
