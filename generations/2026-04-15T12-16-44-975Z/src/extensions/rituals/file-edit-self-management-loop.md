## Process
1. Write or update the target file with the current content or planned changes.
2. Use self-management to reflect on the change, confirm the next action, and keep the work aligned.
3. Read back the relevant file or surrounding context to verify the update and catch errors.
4. Update the todo list to record the completed step and the next pending step.
5. Repeat the write → manage_self → read → todo loop until the file task is complete.
