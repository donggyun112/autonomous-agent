## Process
1. Start by reading the current context or task-relevant material.
2. Use `more_tools` to discover additional tools when the current set is insufficient.
3. Re-read the relevant material after tool discovery to confirm what is needed.
4. If the inspection yields actionable insights, record them in `journal`.
5. Repeat the read/discover/re-read cycle until the required tool or information is identified.
