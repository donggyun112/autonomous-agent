## Process
1. Read the current source material or context carefully.
2. Re-read key passages when the information is dense, ambiguous, or likely to affect later actions.
3. Compare what you found against existing notes or remembered state.
4. Use memory management to add, refine, or prune durable facts when the situation has changed.
5. Record a brief journal entry when a notable update, decision, or continuity point should be preserved.
6. Repeat the read → compare → memory update loop until the state is clear enough to proceed.
