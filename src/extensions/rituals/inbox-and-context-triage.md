## Process
1. Check the inbox first when there may be new user input, updates, or pending items.
2. Read the most relevant current context or reference material.
3. If the next step is unclear, use self-recall to restate the goal and identify what information is missing.
4. Use `more_tools` to surface additional actions or capabilities when the current tool set is insufficient.
5. Re-read the needed material after narrowing the scope to confirm details before acting.
