## Process
1. Check the inbox for new items or pending follow-ups.
2. If the inbox reveals anything requiring more context, open additional tools to inspect the relevant task, note, or thread.
3. Journal concise, durable notes for items that matter beyond the current session.
4. Manage memory to merge duplicates, update existing entries, and remove stale or redundant information.
5. Repeat inbox and memory review until new information is exhausted and the state is clean.
