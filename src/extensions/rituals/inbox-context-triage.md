## Process
1. Start by checking the inbox to surface any new tasks, messages, or priorities.
2. Read the most relevant referenced content to understand the immediate context.
3. Use shell for any deeper inspection, verification, or quick file/system checks that the read step suggests.
4. If needed, repeat inbox checking after journaling to validate whether the captured context has changed.
