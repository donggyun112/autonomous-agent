## Process
1. Check the inbox for new items or updates.
2. Read the relevant content or context needed to understand the item.
3. Decide whether the item requires a memory update, follow-up, or no action.
4. If it requires tracking, update memory with the key facts, decisions, or next steps.
5. Re-check inbox and context as needed until the item is resolved or fully captured.
