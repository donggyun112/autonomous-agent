## Process
1. Check the inbox or queue for new items to process.
2. Record the relevant item, decision, or next action in the journal.
3. Update memory to reflect any durable facts, priorities, or task state.
4. Reconcile memory entries with the journal to remove duplicates, merge overlaps, and keep the active state concise.
5. Repeat inbox review, journaling, and memory maintenance until the queue is clear or the session objective is reached.
