## Process
1. Use `memory_manage` to update or consolidate relevant facts as new information arrives.
2. Periodically `recall_recent_journal` to refresh context before making changes or decisions.
3. Capture important outcomes, commitments, or next actions in `journal`.
4. Translate actionable items from the journal into `todo` entries so they are tracked explicitly.
5. Repeat the memory update cycle when new information supersedes prior context.
