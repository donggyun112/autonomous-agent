## Process
1. Check the inbox first to capture any new instructions or pending items.
2. Read the relevant source content or files to verify context before making changes.
3. Update memory with the confirmed information, avoiding duplicate or conflicting entries.
4. Re-run memory management after any new read if the information changes the state or priorities.
5. Use shell only when needed to inspect or confirm file contents, then feed verified results back into memory.
