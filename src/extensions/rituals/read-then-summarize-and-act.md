## Process
1. Read the current source/materials carefully, then read again to confirm details or compare against prior context.
2. Recall relevant memories or prior decisions when the reading suggests missing context.
3. If the next step requires concrete action, move from reading to shell execution instead of continuing to re-read.
4. Keep the loop tight: read → recall context → act, only repeating reads when they materially change understanding.
