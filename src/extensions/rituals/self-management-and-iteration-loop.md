## Process
1. Write the current state, goal, and next intended action into a durable note or file.
2. Update self/task metadata to reflect the latest priorities, constraints, and progress.
3. Read back the note or relevant context to re-anchor before continuing.
4. Refresh the todo list to capture immediate next steps.
5. Repeat the write → manage_self → read → todo cycle whenever context changes or a work segment completes.
